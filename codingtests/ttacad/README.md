### This exercise covers the following

App takes in a person's height and weight and then guesses if that person likes dogs or cats better.

### Original exercise description

Your application should include the following:

- Your model's guesses should be based the sample data provided for height and weight and pet preference
- Be able to save guesses with each use
- Confirm with the user if your guess was correct
- Guesses should improve with over time with use
- Tests! (testing is very important to us)

Please dive deeper into an area and write a quick summary of what you chose to do (people have chosen things like the algorithm, the front-end via graphs/way for people to explore the data, testing, etc)
